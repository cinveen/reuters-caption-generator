# How to Share This App With a Colleague

## Quick Steps

1. **Zip the entire folder**
   - Right-click the `reuters-caption-generator` folder
   - Select "Compress"
   - This creates a `.zip` file

2. **Send the zip file to your colleague**
   - Via email, Slack, AirDrop, etc.
   - They should have everything they need!

3. **Tell them to:**
   - Unzip the folder
   - Read `SETUP_INSTRUCTIONS.md`
   - Double-click `setup.command` (one time setup)
   - Double-click `start.command` to use the app
