# Reuters Caption Generator - Setup Instructions (Windows)

## For First-Time Users on Windows

Hey! This is a simple app to help you create Reuters-style photo captions using voice recording.

### Prerequisites

You need **Python 3** installed on your Windows PC. If you don't have it:
1. Download from: https://www.python.org/downloads/
2. **IMPORTANT:** During installation, check the box that says "Add Python to PATH"
3. Complete the installation
4. Restart your computer

### Setup (One Time Only)

1. **Extract the zip file** to your Desktop or Documents
2. **Open the `setup-windows` folder**
3. **Double-click `setup.bat`**
   - If you see a security warning "Windows protected your PC", click "More info" → "Run anyway"
   - Let it run (takes 5-10 minutes to download everything)
   - Wait for "Setup Complete!" message

### Running the App

1. **Double-click `start.bat` (in the setup-windows folder)**
2. A window opens with the Reuters Caption Generator
3. Start recording your photo descriptions!

### Using the App

1. **Click "Start Recording"** and describe your photo
2. **Click "Stop Recording"** when done
3. **Click "Generate Reuters Caption"** to get a formatted caption
4. **Copy the caption** to use in your workflow

### Troubleshooting

**"Python is not installed" error:**
- Install Python 3 from https://www.python.org/downloads/
- **Make sure to check "Add Python to PATH"** during installation
- Restart your computer after installing

**Windows Defender warning:**
- Click "More info" → "Run anyway"
- This is normal for scripts that aren't from the Microsoft Store

**"Access Denied" error:**
- Right-click on setup.bat → "Run as administrator"

**App won't start:**
- Make sure you ran `setup.bat` first
- Check that Python 3 is installed: Open Command Prompt and type `python --version`
