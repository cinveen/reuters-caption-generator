# Reuters Caption Generator - Setup Instructions (macOS)

## For First-Time Users on Mac

Hey! This is a simple app to help you create Reuters-style photo captions using voice recording.

### Prerequisites

You need **Python 3** installed on your Mac. If you don't have it:
1. Download from: https://www.python.org/downloads/
2. Install it (keep clicking "Next")
3. Restart your computer

### Setup (One Time Only)

1. **From the main folder, go to the `setup-mac` folder**
2. **Double-click `setup.command`**
   - If you see a security warning, right-click → Open → Open anyway
   - Let it run (takes 5-10 minutes to download everything)
   - Wait for "Setup Complete!" message

### Running the App

1. **Double-click `start.command` (in the setup-mac folder)**
2. A window opens with the Reuters Caption Generator
3. Start recording your photo descriptions!

### Using the App

1. **Click "Start Recording"** and describe your photo
2. **Click "Stop Recording"** when done
3. **Click "Generate Reuters Caption"** to get a formatted caption
4. **Copy the caption** to use in your workflow

### Troubleshooting

**"Permission Denied" error:**
- Right-click on `setup.command` → Open → Open anyway

**Security warning about unknown developer:**
- Go to System Preferences → Security & Privacy
- Click "Open Anyway" at the bottom

**App won't start:**
- Make sure you ran `setup.command` first
- Check that Python 3 is installed
